# SimpleNote

### SimpleNote는 할 일을 간편하게 기록할 수 있는 앱이에요.
- iCloud를 통해 iPhone, iPad 등 애플 생태계의 모든 앱 연동되어요.
- 폴더를 생성하여 원하는 할 일만 볼 수 있어요.
- 요일별로 나의 할 일을 손쉽게 확인할 수 있어요.
